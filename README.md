EduHub is a high-performance, AI-driven course aggregation platform designed to simplify the discovery of competitive exam resources. Inspired by Apple's premium design language, it offers a stunning 3D user experience with glassmorphic elements, intent-based search, and a self-learning AI assistant.
 Key Features
 1. Premium 3D User Experience
- Apple-Inspired Design: Sleek dark mode, glassmorphism, and "SF Pro" typography.
- 3D Tilt Cards: Interactive course cards that react to mouse movement with depth and reflection effects.
- Ambient Glows: Dynamic background gradients that create a premium, immersive atmosphere.

 🤖 2. Self-Learning AI Chatbot (EduHub AI)
- Intent Recognition: Understands user goals (e.g., "Find free UPSC courses").
- Memory Persistence: Remembers past searches and interests to provide personalized greetings and recommendations.
- Real-time Discovery: Directly indexes 80+ courses to provide instant details on instructors, ratings, and pricing.

 3. Intelligent Global Search & Filtering
- Persistent Search: A global search system that works across the hero section and navbar.
- Advanced Filters: Real-time filtering by Exam Category (JEE, NEET, UPSC, SSC, etc.), Price, Rating, and Language.

4. Massive Course Aggregation
- 80+ Courses: Pre-populated data covering 40+ exam categories.
- Multi-Platform Sync:Consolidates data from top providers like Physics Wallah, Unacademy, Byju's, and more.

 📂 Project Structure

- ux_sample.html: The core platform—a highly optimized, single-page application (SPA).
- generate_all_courses.py: The "Data Engine" that populates the UI with the latest course datasets.
- scrape_pw.py: Automated scripts for scraping real-time course data from Physics Wallah.
- fix_html.py: Optimization scripts to ensure the UI remains fluid and bug-free.
